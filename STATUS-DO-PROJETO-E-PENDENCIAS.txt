============================================================
3. COMO INSTALAR / ATUALIZAR
============================================================

1. Fazer backup do projeto atual no Replit.
2. Substituir os arquivos pelos deste pacote.
3. Manter as Secrets existentes:
   - SUPABASE_URL
   - SUPABASE_PUBLISHABLE_KEY
4. Não adicionar service_role ou Secret Key no frontend.
5. Abrir Supabase > SQL Editor > New query.
6. Executar todo o arquivo supabase/migration_v2.sql.
7. Confirmar Success.
8. No Replit, executar:
   npm install
   npm run check
   npm start
9. Testar no Preview.
10. Confirmar profiles.role = owner para a conta correta.
11. Testar login Google e Discord.
12. Testar catálogo, carrinho, Pix, pedidos, chat, equipe e avaliações.
13. Somente depois clicar em Republish.
